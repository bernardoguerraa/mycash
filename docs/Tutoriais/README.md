# Tutoriais

> Material de aprendizado guiado. Para quem está começando no projeto.

Tutoriais conduzem o leitor por uma sequência de passos com resultado garantido.
São voltados para **aprendizado**, não para resolver um problema específico.

## Conteúdo planejado

- `01-primeiro-setup.md` — clonar o repositório, configurar `.env.local`, rodar `npm install`, subir o projeto local
- `02-primeira-feature.md` — implementar uma feature simples ponta a ponta (ex.: adicionar campo numa transação)
- `03-fluxo-de-dados.md` — entender como os dados fluem do Supabase ao componente React

## Característica de um bom tutorial

- O leitor segue os passos sem precisar tomar decisões
- Cada passo tem um resultado verificável
- Foca no que **funciona**, evita explicar o porquê (isso é Explanação)
